# BandBot

Telegram-бот для управления репертуаром музыкальной группы.

## Возможности

- Управление песнями: тональность, темп, ответственный, примечания, закрепы
- Сетлисты с fuzzy-поиском песен
- Подписки на изменения и уведомления
- История изменений
- Inline-поиск песен

## Запуск

```bash
cp .env.example .env
# Заполните BOT_TOKEN в .env
docker compose up -d
```

## Команды

| Команда | Описание |
|---------|----------|
| `/song Название` | Показать или создать песню |
| `/song Название: 120 Cm` | Создать/обновить с темпом и тональностью |
| `/setlist Название` | Показать или создать сетлист |
| `/setlist` | Показать активный сетлист |
| `/all_songs` | Список всех песен |
| `/all_setlists` | Список всех сетлистов |
| `/toggle_subscribe_all` | Подписаться на все изменения в чате |
| `/help` | Справка |

## Хештеги

```
#примечание Название песни
Текст примечания
```

```
#закреп Название песни
Название закрепа
```
(ответом на закрепляемое сообщение)
